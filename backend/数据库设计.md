# 数据库表结构设计

## 1. 用户表 users

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| phone | VARCHAR(20) | UNIQUE, NOT NULL | 手机号 |
| password | VARCHAR(256) | NOT NULL | 加密存储 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | |
| updated_at | DATETIME | DEFAULT CURRENT_TIMESTAMP ON UPDATE | |

## 2. 成功日记表 diaries

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| user_id | INT | FK(users.id), NOT NULL, INDEX | 所属用户 |
| content | TEXT | NOT NULL | 日记内容，≤2000 字符 |
| date | VARCHAR(10) | NOT NULL | 日期 yyyy-MM-dd |
| category | VARCHAR(32) | NOT NULL | work/health/relationship/growth/daily |
| mood_icon | VARCHAR(64) | | 情绪图标 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | |
| updated_at | DATETIME | ON UPDATE | |

## 3. 美德践行记录表 virtue_logs

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| user_id | INT | FK(users.id), NOT NULL, INDEX | 所属用户 |
| virtue_type | VARCHAR(32) | NOT NULL | 美德类型 |
| completed | TINYINT(1) | NOT NULL, DEFAULT 1 | 是否完成 |
| reflection | VARCHAR(2000) | | 感想，≤2000 字符 |
| date | VARCHAR(10) | NOT NULL | 日期 yyyy-MM-dd |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | |
| updated_at | DATETIME | ON UPDATE | |

美德类型枚举：友好亲和、勇于担当、善待他人、帮助给予、感恩之心、勤学不辍、值得信赖

## 4. 愿景板表 vision_items

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| user_id | INT | FK(users.id), NOT NULL, INDEX | 所属用户 |
| category | VARCHAR(32) | NOT NULL | work/health/relationship/growth/daily |
| title | VARCHAR(128) | NOT NULL | 标题 |
| description | VARCHAR(512) | | 描述 |
| image_url | VARCHAR(512) | | 图片 URL（COS 或 asset:xxx） |
| target_date | VARCHAR(32) | | 目标日期 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | |
| updated_at | DATETIME | ON UPDATE | |

## 5. 枚举值

- **diary/vision category**: work, health, relationship, growth, daily
- **virtue_type**: 友好亲和, 勇于担当, 善待他人, 帮助给予, 感恩之心, 勤学不辍, 值得信赖
